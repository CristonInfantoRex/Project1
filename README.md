# FLAMES Love Calculator

A modern, interactive one-page FLAMES website with smooth animations and glassmorphism UI.

## Features

- Fullscreen background video
- Smooth slide transitions
- Glassmorphism design
- FLAMES algorithm calculation
- Animated result display with floating emojis
- Mobile responsive

## Setup

1. Place your background video file as `background.mp4` in the root directory.
   - The video should be suitable for looping (e.g., romantic scenery, hearts, etc.)
   - Recommended: MP4 format, muted, auto-play friendly

2. Open `index.html` in a web browser to run the website.

## Files

- `index.html`: Main HTML structure
- `styles.css`: CSS styling with glassmorphism and animations
- `script.js`: JavaScript logic for FLAMES calculation and interactions

## Customization

- Replace `background.mp4` with your own video file
- Adjust colors, fonts, and animations in `styles.css`
- Modify the FLAMES algorithm or add features in `script.js`

## Browser Support

- Modern browsers with support for CSS backdrop-filter and video autoplay
- Mobile responsive design
